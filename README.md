# Developer & Creative Portfolio Hub

A centralized, responsive portfolio architecture split into dual development contexts (**Technical Workspaces** and **Creative Hubs**). The web engine utilizes modular glassmorphic blocks paired with interactive hardware-accelerated video background layers that toggle fluidly based on the user's focus.

## 🛠️ Technology Stack Matrix

* **Frontend Engine:** Semantic HTML5, CSS3 Custom Properties Architecture, Vanilla ECMAScript 6+ Systems.
* **Layout Framework:** Bootstrap v5.3.3 Grid & Core Utility Classes.
* **Dynamic Data Integrations:** Native Asynchronous JavaScript (Fetch REST Architecture) referencing the GitHub API.
* **Icon Distribution:** FontAwesome v6.4 Webfonts & Devicon Vector Assemblies.

## 📦 Dynamic Repositories Overview
The portfolio relies on standard runtime components to execute asynchronous handshakes tracking external codebase states.

### Core Repository Stack Highlights
* **Dynamic Code Injection:** Automatically captures open repository entities, filters out structural configurations, strips nomenclature boundaries (hyphens/underscores), and formats responsive UI blocks tracking languages used.
* **Carousel Controller Engine:** Custom structural sliding boundaries that compute element bounding widths dynamically. Supports safe fallbacks if the user's connection drop prevents populating cards.
